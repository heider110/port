Project Details

Crypto Wallet is a Web-Application to monitor Crypto currency prices and calculate your long-term profits and losses. All you have to do is register an account either via your e-mail or Google authentication and then create your own wallet and start to enter the buying or selling transactions manually or retrieve them from the platforms through which you made the purchases through your own API in a very easy and smart way, and the system will calculate the profits and losses for each transaction and for each currency in general as it calculates the average buy price and compares it to the current price. The return on investment ROI and more features will be also calculated. The application currently supports only Binance exchange, however i will add other platforms with the next updates.
